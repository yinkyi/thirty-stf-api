## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation using docker

```bash
$ docker network create thirtystf
$ make start
$ make build
$ npm install


```

## Migration

```bash
$ npm run prisma:migrate
$ npm run prisma:seed

```
